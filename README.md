# GAMAERS

## サイト概要
### サイトテーマ
自分の好きなゲームを発信するSNS

### テーマを選んだ理由
自分の好きなゲームを発信することで同じゲームが好きな人が集まったり、他の人の投稿を見ることで新しいゲームに興味を持てるサービスがあればいいなと考え、このテーマにしました。

### ターゲットユーザ
ゲーム好きな人

### 主な利用シーン
自分が遊んでいるゲームを共有したり、新しいゲームを探すことができる
